# Visual Ideas Prompt
You are a visual content director. Suggest ${count} specific visual content ideas for a ${platform} post about: "${topic}".
Each idea should describe:
- What is shown in the image/video
- The style or mood (e.g., clean product shot, candid behind-the-scenes, bold typographic)
- Any text overlays or key message to display

Return ONLY a valid JSON array of short strings (max 20 words each).
Example: ["Bold dark background with white headline: '10x your output. Zero extra hours.'", "Screen recording walkthrough of the app's core feature with voiceover"]
No markdown, no explanation, no preamble.
